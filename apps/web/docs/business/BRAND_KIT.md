## 1. Brand Overview

**BuildOS** is an AI-powered productivity platform that transforms scattered thoughts into organized action. Built for disorganized minds who need to get organized.

**Core Metaphor:** Inkprint - a printmaking-inspired aesthetic with semantic textures, warm accent colors, and high information density.

**Tagline Options:**
- "Project organization built for the AI era."
- "Your thoughts, organized. Your next step, clear."
- "Your brain isn't broken. Your tools are."
- "From brain dump to organized projects in literally one minute."

## 5. Brand Voice

### Personality Traits

| Trait | Description |
|-------|-------------|
| **Empathetic** | We understand the struggle. No judgment. |
| **No-Shame** | "Your disorganization isn't a personal failure. It's a tool problem." |
| **Direct** | Clear, concise, no fluff |
| **Confident** | We know our solution works |
| **Playful** | Light-hearted, not corporate |

### Tone Guidelines

| Context | Tone | Example |
|---------|------|---------|
| **Marketing** | Empathetic, inspiring | "Your brain isn't broken. Your tools are." |
| **Onboarding** | Encouraging, simple | "Brain dump everything. We'll organize it." |
| **Error Messages** | Helpful, not blaming | "Something went wrong. Let's try again." |
| **Success States** | Celebratory, brief | "Done! One step closer." |
| **Technical/Docs** | Clear, precise | "Click Generate to create your tasks." |
| **Investor** | Professional, data-driven | "AI-native architecture built from day one." |

### Key Messages

**Core Promise:**
> "Project organization built for the AI era."

**Value Propositions:**
1. **Brain Dump → Structure** - Transform unstructured thoughts into organized projects
2. **Calendar Execution** - Schedule tasks directly into your calendar
3. **ADHD-Friendly** - Built for scattered minds, no shame approach
4. **Context Over Chaos** - AI maintains context across all your work
5. **AI-Native** - Built for the AI era, not retrofitted

### Copy Patterns to Use

**Problem Framing:**
- "You've tried the complex systems. You've abandoned the perfect planners."
- "Traditional productivity tools demand linear thinking. Your brain doesn't work that way."
- "When everything feels urgent, nothing is clear."

**Solution Framing:**
- "BuildOS is your home base. The one place where your chaos becomes clarity."
- "Dump thoughts in any order. AI finds the structure."
- "From 'I should' to 'I did'."

**Trust Builders:**
- "14-day free trial. No credit card required."
- "Actually built for ADHD."
- "Made with love for productivity."

## 6. Design Patterns (Inkprint)

### Texture System

BuildOS uses semantic textures inspired by printmaking:

| Texture | Meaning | Usage |
|---------|---------|-------|
| **Bloom** | Ideation, newness, creative expansion | Creation flows, "new project", drafts |
| **Grain** | Execution, steady progress | Active work views, task lists |
| **Pulse** | Urgency, deadlines, momentum | "Today focus", priority zones |
| **Static** | Blockers, risk, warnings | Error states, "needs triage" |
| **Thread** | Relationships, dependencies | Linked entities, shared projects |
| **Frame** | Canon, structure, officialness | Primary containers, modals |

### Shadow System

| Shadow | Usage |
|--------|-------|
| **Shadow-ink** | Standard card elevation, subtle |
| **Shadow-ink-strong** | Modals, overlays, elevated panels |
| **Shadow-ink-inner** | Inputs, inset elements |

### Corner Radius

| Element | Radius |
|---------|--------|
| Buttons | 0.5rem (8px) |
| Cards | 0.5rem (8px) |
| Modals | 0.5rem-0.75rem |
| Inputs | 0.375rem-0.5rem (6-8px) |
| Badges | 0.25rem (4px) |

## 9. Quick Reference - Canva Setup

### Import These Colors

```
Primary Accent: #F97316
Ink Black: #18181B
Paper White: #FAF9F7
Card Light: #F7F5F2
Muted Light: #EDEBE7
Border Light: #DBD8D2
Text Muted: #6B6B73
Success: #059669
Warning: #D97706
Danger: #DC2626
```

### Font Setup

1. **Primary Font:** Inter (Google Fonts)
2. **Secondary Font:** IBM Plex Serif (Google Fonts)
3. **Fallback:** System UI

### Logo Usage Rules

1. Always maintain clear space around the logo (minimum 16px)
2. Use light logo on dark backgrounds, dark logo on light backgrounds
3. Never stretch or distort the logo
4. Minimum size: 80px width
5. "OS" should always be the accent orange color
