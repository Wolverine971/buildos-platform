# LinkedIn Tailoring

Reshape an approved draft into a LinkedIn post. LinkedIn rewards text-first posts that earn the "…see more" expand and drive comments. Carries: text + one image/carousel/document.

## Format constraints

- ~3,000 character limit; strongest posts run **900–1,500 characters**.
- Only the **first ~2 lines (~140 chars)** show before "…see more". That is the entire attention slot.
- One visual asset: a single image, a native PDF "document" carousel, or (less reach) an external-link post — links in-body suppress reach, so put links in the **first comment**.
- No markdown rendering. Use line breaks, not headers. Short paragraphs only.

## Structure template

```
Line 1: hook (≤ ~10 words, pattern-break or curiosity gap)
[blank line]
2–4 short paragraphs, 1–2 sentences each, blank line between
  — front-load the payoff; do not bury the turn below the fold
[blank line]
One-line takeaway / reframe
[blank line]
CTA: a single question that invites a reply
```

## Hook placement

- Move the single strongest beat to line 1. It must work **alone**, before the expand.
- Avoid line-1 throat-clearing ("I've been thinking about…", "Here's a thread on…").
- A specific number, a confession, or a flat contrarian claim out-pulls a clever abstraction.

## Length thresholds

- Hook line: **≤ 10 words**.
- Paragraphs: **1–2 sentences**, never a wall.
- Total: **900–1,500 chars** for a standard post; document carousels carry the overflow.

## Amplification levers

- **Curiosity gap in line 1** that the expand pays off.
- **Reply-prompt CTA** — comments weight reach more than likes; ask one concrete question.
- **First-comment link** for any external URL.
- **Document carousel** for step-by-step or list content (high dwell, high saves).
- Reply to early comments fast to compound the post's first-hour velocity.

## CTA norms

- One primary CTA: a question that is cheap to answer ("What's the longest…", "Which one are you?").
- Do not stack "follow me + subscribe + link in comments". One ask.

## Common mistakes

- Hook hidden below the "…see more" fold.
- A link in the body (reach penalty) instead of the first comment.
- A wall of text with no line breaks.
- Multiple CTAs that split the reader's attention.
- Generic motivational close with no specific reply-prompt.
