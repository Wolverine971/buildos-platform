# Foundation Rules: Site-Wide Pre-Conditions and Cross-Cutting Layout Principles

Use this reference when starting any marketing-site review, or when the user's concern is overall visual quality ("why does my site look amateur/templated?") rather than one section. These pre-conditions must hold across the whole site before section-level review is meaningful — cite the specific rule violated in each finding.

## Foundational rules (apply everywhere)

- **4-pixel system.** Every spacing, padding, gap, container width, and font size is a multiple of 4px (scale: 4, 8, 12, 16, 24, 32, 48, 64, 96). Type scale: 16, 20, 24, 28, 32, 40, 48, 64. Reject off-scale values (13px, 27px, "looks about right").
- **Type roles only.** Headings (2–3 levels), paragraphs, buttons, labels. No invented styles per section.
- **Line height inversely proportional to size.** ~1.0× for big headlines, ~1.5× for body. Never trust browser 125%.
- **Body text contrast ≥ 4.5:1**, ideally 7:1.
- **60/30/10 color proportion.** 60% dominant, 30% secondary, 10% accent (CTAs only). Complementary colors never split 50/50.
- **Hierarchy ranking.** One primary, a few secondary, the rest uniform — per section _and_ across the page.
- **Layout effort uniform.** Hero, FAQ, testimonials, contact all get the same care. Dropping effort in any one section signals "template." Effort drop-off is the strongest amateur signal.

## Cross-cutting layout principles

- **Two zones beats four.** One message per hero, per section.
- **Break the grid intentionally.** Rigid grids read as templated; floating composition reads as bespoke.
- **Full-width sections** for testimonials, sub-heroes, and demos signal importance.
- **Vary tile sizes.** Bento-box grids beat 3×3 uniform grids for features.
- **Real visuals beat stock visuals**, every time, in every section.
- **Layout effort must be uniform** across sections — score it explicitly (1–5) in the roll-up and flag any section that visibly received less effort.
