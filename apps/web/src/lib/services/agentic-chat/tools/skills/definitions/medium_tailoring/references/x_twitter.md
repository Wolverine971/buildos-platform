# X / Twitter Tailoring

Reshape an approved draft into a single X post or a thread. X rewards a hard hook and one idea per unit. Carries: short text, images, video, and threaded continuation.

## Single post vs thread

- **Single post** — one sharp idea, a confession, a contrarian line. Default for a small framed idea.
- **Thread** — a sequence the spine already implies (steps, a story, a list). Use only when each tweet earns its place.

## Format constraints

- Standard post: **280 characters** (longer for premium, but the 280 discipline reads better).
- Thread: the **first tweet is the whole hook** and must stand alone; it is what gets quoted and reposted.
- Up to 4 images per tweet; a single bold image lifts a text tweet.
- Links suppress reach — put the link in a **reply**, not the hook tweet.

## Structure template (thread)

```
Tweet 1  Hook — the strongest claim/payoff, standalone, no "a thread 🧵" filler
Tweet 2  Stakes or the turn — keep them reading
Tweets 3–n  One beat per tweet, map the spine 1:1; each tweet readable alone
Final    Payoff + one CTA (follow / bookmark / reply); link in a separate reply
```

## Hook placement

- Tweet 1 carries the entire job. Lead with the result, the number, or the reversal.
- Cut "I'm going to explain…" framing — state the thing.
- A line that invites disagreement or recognition earns quote-tweets.

## Length thresholds

- Hook tweet: **≤ ~200 chars** so it is fully visible and quotable.
- Body tweets: **one idea each**; if a tweet needs "and also", split it.
- Thread length: **5–9 tweets** for most; longer only if every tweet is load-bearing.

## Amplification levers

- **Quote-bait**: a claim people want to react to (agree hard / push back).
- **Bookmark-bait**: a tight list or framework people save for later.
- **Reply-prompt** final tweet; reply to early responses to lift velocity.
- Standalone tweets so any single one can go wide out of context.

## CTA norms

- One ask: follow for more, bookmark, or reply. Link goes in a reply, never the hook.

## Common mistakes

- "🧵 A thread on…" preamble instead of a real hook.
- A link in tweet 1 (reach penalty).
- Tweets that only make sense in sequence (kills repost-ability).
- Stuffing several ideas into one tweet.
- A thread where tweets 3–7 are filler the idea did not need.
