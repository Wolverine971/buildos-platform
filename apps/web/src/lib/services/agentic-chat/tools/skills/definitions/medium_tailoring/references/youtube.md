# YouTube Tailoring

Reshape an approved draft into a YouTube video (long-form) or Short. On YouTube the **packaging** (title + thumbnail) does most of the work, and the first 30 seconds decide retention. Carries: video + audio + on-screen text + description.

## Long-form vs Short

- **Long-form** — explanation, story, teardown, walkthrough. Packaging + retention structure are the job.
- **Short** — one hook-driven beat, vertical 9:16, ≤60s. Same hook rules as a Reel.

---

## Packaging (the highest-leverage surface)

- **Title and thumbnail are designed together** as one promise; never write them separately.
- Title: **≤ ~60 characters** so it is not truncated; lead with the outcome or the curiosity gap.
- Thumbnail: **one focal idea**, ≤ 3 words of text, readable at phone size. The image and title should _not_ repeat each other — they should combine.
- Run the CCN check (Core / Casual / New): would the packaging pull a viewer with no context? If it only lands for insiders, the ceiling is the existing audience — repackage for a Casual viewer.

## Intro (first 30 seconds = retention cliff)

```
0–5s    Restate the title's promise in motion; show, don't preamble
5–15s   Stakes — why this matters / what they'll be able to do
15–30s  Roadmap or first payoff; no long channel intro, no "smash like" yet
```

## Retention structure (body)

- Map the spine's beats to **segments**; open each segment with a mini-hook (a rehook) and a reason to stay.
- Use **open loops**: tease a later payoff early, close it on time.
- Sync `sensory_double_tap` cues as on-screen `[VISUAL: …]` / b-roll at each spoken claim.
- Cut dead air; every 20–30s should advance or re-hook.

## CTA placement

- Earn the CTA: place the main ask **after** the first real payoff, not in the cold open.
- One primary ask (subscribe _or_ the next video). End screen → the logical next video, not a generic playlist.

## Thresholds

- Title ≤ ~60 chars; thumbnail text ≤ 3 words.
- Intro hook delivered in ≤ 30s; first payoff before the 1-minute mark.
- One primary CTA, placed post-payoff.

## Common mistakes

- Writing the title without the thumbnail (or vice versa).
- A slow talking-head intro that burns the retention cliff.
- Packaging that only Core viewers understand (no breakout ceiling).
- "Like and subscribe" before delivering any value.
- Thumbnail text that just repeats the title.
