# Visual Craft Fundamentals

Use this child when the screen functions but does not feel crisp, intentional, or professionally designed.

## When to Use

- The UI looks generic, amateur, overly dense, visually flat, noisy, or AI-generated.
- The user asks for polish, taste, premium feel, visual craft, spacing, type, color, shadows, or layout refinement.
- A prior UI/UX review found no major flow issue but the interface still lacks quality.

## Workflow

1. Establish the focal hierarchy: primary content, primary action, secondary controls, and background structure.
2. Normalize spacing and alignment using a small spacing scale and consistent section rhythm.
3. Tune typography by role: display, heading, body, label, metadata, helper text, and error text.
4. Repair color and contrast roles before adding decoration. Use neutral structure, one primary action color, semantic colors, and restrained accents.
5. Polish depth, borders, icons, states, and empty areas only after structure, type, and spacing are working.

## Guardrails

- Do not add decorative gradients, blobs, or oversized cards as a substitute for hierarchy.
- Do not use color as the only carrier of meaning.
- Do not make a compact tool surface feel like a marketing hero.
- Keep recommendations implementable in the existing design system.
