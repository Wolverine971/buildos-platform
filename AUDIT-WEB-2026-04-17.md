# BuildOS Web App Audit тАФ 2026-04-17

Scope: `apps/web` full audit тАФ configuration, dependencies, pages/routes, code correctness, security. Parallel audit run by three Explore agents.

**Overall assessment: A-.** Svelte 5 migration 99.7% complete, ApiResponse usage near-universal, strong auth coverage. Real risks concentrated in dependency hygiene, silent error swallowing, and one admin-surface XSS.

## ЁЯФ┤ CRITICAL тАФ Fix immediately

### C1. SvelteKit security CVEs (2 advisories) тАФ **FIXING NOW**

- `apps/web/package.json:75` тАФ `@sveltejs/kit@^2.57.0`.
- **CVE 1**: `adapter-node` BODY_SIZE_LIMIT bypass (GHSA-2crg-3p73-43xp, HIGH, patched in 2.57.1).
- **CVE 2**: Unvalidated redirect in handle hook тЖТ DoS (GHSA-3f6h-2hrp-w5wx, MODERATE, patched in 2.57.1).
- Fix: bump to `^2.57.1`. Update root `pnpm.overrides` entry from `@sveltejs/kit@<2.49.5` to `@sveltejs/kit@<2.57.1`.

### C2. XSS in admin email history viewer тАФ **FIXING NOW**

- `apps/web/src/lib/components/admin/EmailHistoryViewerModal.svelte:89` тАФ `{@html email.html}` with no sanitization.
- Impact: admin session hijack via malicious email stored in DB.
- Fix: run through `sanitize-html` before rendering. `sanitize-html` is already a direct dep.

### C3. `zod` in wrong dependency bucket тАФ **FIXING NOW**

- `apps/web/package.json:120` тАФ in `devDependencies` but used at request time in route handlers and services.
- Impact: breaks if devDependencies stripped in any build variant.
- Fix: move to `dependencies`.

### C4. Transitive vulnerabilities тАФ **FIXING NOW**

Current `pnpm audit`:

- `follow-redirects тЙд1.15.11` leaks custom auth headers (via `twilio > axios`). Moderate. Need override.
- `path-to-regexp <0.1.13` already overridden in root тЬУ but worker uses express 4.21.2 тЖТ 0.1.12 transitively тАФ override reinforcement needed.
- `webpack <5.104.1` via `@antv/g6` (LOW, SSRF build-time). Override or deprecate g6 subdeps.
