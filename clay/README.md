## Data Structure

Each CSV file contains the following columns:

- **Name**: Contact's full name
- **Company Name**: Current organization or business
- **Job Title**: Professional title or role
- **Location**: City, State, Country
- **LinkedIn URL**: Direct link to LinkedIn profile

## Geographic Focus

All 295+ contacts are based in the Maryland/DC/Baltimore corridor, making this list ideal for:

- Local outreach campaigns
- Regional partnership development
- Community building initiatives
- Event speaker recruitment

## Usage Notes

- **Data Quality**: Some entries have incomplete information (marked with "(No Job Title)" or "(No Company)"). These are valid records but may require verification.
- **Duplicates**: Some names may appear in multiple lists (this is intentional - they qualified for multiple categories)
- **LinkedIn URLs**: All URLs are current as of the export date and link directly to professional profiles
- **Locations**: Locations have been standardized but preserve the original data as exported from Clay
