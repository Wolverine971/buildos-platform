## Beat skeleton

Hook ("cheat sheet") → problem ("everyone looks the same") → thesis (X = emotion) → **6× [info card (Use when / Best for / Creates) + demo image tagged Emotion=X], interleaved with periodic Quick-rule decision slides** → **"Save this formula" recap card** → "which do you use?" comment CTA.

This is a different engine from the Trio→Venn decks: it's an **enumerated reference cheat sheet**, where the value is the comprehensive lookup table, and the "Save this formula" recap is the payload. Notice the **info-card sub-template** repeated 6×: a heading, "Use when," "Best for" tags, and "Creates" emotion tags.

## Highest-value extractable — the cheat-sheet template

This is the single best model for a **9takes "Type → Core Emotion" cheat sheet**, because the Enneagram already IS a 6-(well, 9-)item taxonomy:

- Re-skin the **info card** as: **Type #N · Use when [you see this behavior] · Best for [recognizing in...] · Driven by [core emotion tags].**
- Re-skin **"Emotion = X"** demo tag as **"Core fear = X"** or **"What they want = X."**
- The **"Save this formula" recap** becomes a one-slide **9-type → core-motivation lookup** — guaranteed saves.
- Keep the **"Quick rule" decision slides** ("Want X? → do Y") as "Want to connect with a Type 8? → don't flinch. Want to lose them? → get vague."

## 9takes translation

"**The Enneagram cheat sheet: which type feels which emotion.**" One card per type (Use when / recognize by / driven by), demo tag "Core fear = \_\_\_", and a final save-this 9-type recap. This is the most directly liftable structure in the whole set for an evergreen, save-bait reference carousel. Amber palette; the recap slide is the light "peak" slide.
