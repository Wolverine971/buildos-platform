# Import into HyperAgent

This bundle contains no credentials, customer data, schedules, channels, or external integrations.

1. In HyperAgent, open **Settings**.
2. Choose **Import from OpenClaw**.
3. Upload `buildos-visual-director-hyperagent-import.zip`.
4. Review the preview. Keep the identity, system prompt, memories, and tool guidance.
5. Finish the import with schedules and Live Mode disabled.
6. In the imported agent's **Skills** settings, pin `hyperframes`, `hyperframes-cli`, and `website-to-hyperframes`.
7. Keep `gsap` and `hyperframes-registry` available but unpinned.
8. Open a new thread and upload the Cold Manuscript source package.
9. Paste `HYPERAGENT_THREAD_PROMPT.md` from that project.

The import creates the persistent agent. Each new asset should still get its own thread.
