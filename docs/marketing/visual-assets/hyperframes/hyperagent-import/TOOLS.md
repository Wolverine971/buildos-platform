# Tools and Skills

## Pin these skills

- `hyperframes`
- `hyperframes-cli`
- `website-to-hyperframes`

## Keep available, not pinned

- `gsap`
- `hyperframes-registry`
- `video-prompting` only when editing supplied real footage

## Do not attach

- `advanced-image-techniques`
- image-generation tools
- avatar-video tools
- open-ended generative video tools

## Enable

- code execution;
- file management;
- webpage/browser capture;
- GitHub when the BuildOS repository should be read directly.

## Model behavior

- cheapest capable coding model;
- low or standard effort;
- no subagents;
- smallest practical run budget;
- increase effort only after one specific lint/composition problem survives a correction pass.
