# Colt Kirwan — NYC maker / lifestyle

## Profile

- **Channel:** [@ColtKirwan](https://www.youtube.com/@ColtKirwan)
- **Lifetime views:** 9M.
- **Origin video:** "I actually chased my dream for one year. This is what happened." — flagship hero's-origin story.
- **Current direction:** Pivoting hard into maker / tinker space ("Casey Neistat meets Michael Reeves").
- **Income breakdown:** **~95% from short-form brand partnerships.** YouTube long-form = creative outlet, no ad breaks, "burning money but passion."
- **Self-described:** Wants to be in the "builder / maker / tinkerer space."

## What he's building

A long-form YouTube identity grounded in "I built X" videos — apartments, robots, dream lives. Recent flagship: "I built my dream tiny NYC bedroom." Wants to do "I built a robot to solve my loneliness problem" and similar. Money currently comes from short-form sponsored content; long-form is the brand-building / community layer.

## Colin & Samir's diagnosis

1. **TAM analysis** — biggest possible audience for each idea.
2. **Engine format = "I built X"**. Lock it in. All long-form fits this.
3. **Tiny stories with Colt Kirwan** as a potential short-form format. Tools / objects → bigger story.
4. **Short-form must move into a recurring sponsored show** so brand inbound isn't ad-hoc.
5. **Don't compose thumbnails — find them in real frames.** Subway Takes model.
6. **Make the personal stuff fit "I built X"** — "I built my dream body" (90-day transformation), "I built a movie reel for my wife."

## Tool / workflow signal

- **Build-then-record loop** — each long-form requires a physical thing to be built. Months of work per video.
- **Short-form sponsor portfolio.** ~90% of income. Probably manages with email + spreadsheet, no CRM mentioned.
- **Ideas live in his head** — sent C&S "a couple of projects I have in mind" but no system surfaced on-camera.
- **"What I built" gallery** — the channel itself is a portfolio that buyers reference. Each build is a project.

## BuildOS angle

Colt is the indie-tutorial-creator-meets-maker archetype. Every long-form is a multi-month physical-build + research + film cycle. The robot, the loft, the desk, the dream body — each is a project that needs design notes, research, time-lapse footage, sponsor placements, and a "what I learned" reflection that compounds into the next build. Right now those projects don't talk to each other — and his short-form sponsored format (the 95% revenue side) is ad-hoc, not a packaged offering.

**Pitch shape:** "Your engine is 'I built X.' Each build is a 3-6 month project — research, sourcing, time-lapse, sponsor placements, the eventual long-form. Right now each build starts from zero. BuildOS holds one project per build, so when video #7 has you building a robot, you can reference what you learned about lighting in the loft bed and what sponsor worked best on the desk. The maker space is where this matters most — every project has tools, materials, parts orders, and learnings."

- 95% of revenue is short-form sponsored — frame BuildOS as protecting the long-form passion side, not optimizing the short-form revenue side.
- Vintage-Casey-Neistat aesthetic preference. Visual / craft-respecting pitch matters.
- He referenced Van Neistat building a movie reel for Samir's wedding — references Studio practice / "show your work." Right voice.
