## Status legend

- **Profiled** — full Stage-1 dossier exists.
- **Candidate** — captured in the pipeline backlog; needs data re-verify + profile before send.
- **Listed** — named on a thesis/campaign list, not yet profiled.
- **Skip / orbit** — deliberately not cold-pitched (engage organically only).
- **Partner-lane** — partnership/integration motion, NOT a user/creator pitch.

## How to add a new target

1. Search this page for the name. Already here → go to the owning doc.
2. Not here → add to the **candidate backlog** in `CANDIDATE-PIPELINE-2026-06-19.md` (with the verify-before-send caveats), then add a one-line row/entry to the relevant section above.
3. When a candidate gets a full dossier, move it to **Profiled** here and into `ai-native-builders/` (builder/validator) or `creators/` (educator/connector/creator).
