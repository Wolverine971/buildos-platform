# Lead Campaigns

This folder is the working set for active BuildOS lead-generation campaigns.

Use it to keep campaign work narrow, measurable, and reviewable.

---

## File Types

### Campaign brief

Format:

`YYYY-MM-DD_<campaign-slug>_campaign.md`

Use this for:

- segment choice
- pain thesis
- proof asset
- CTA
- channel plan
- scorecard
- next 7 actions

### Campaign check-in

Format:

`YYYY-MM-DD_<campaign-slug>_checkin.md`

Use this for:

- weekly progress
- metrics
- learnings
- verdict
- next actions

---

## Rules

- Prefer one active campaign at a time.
- Two active campaigns is the hard maximum.
- Every campaign must have one primary segment and one primary CTA.
- Every campaign must end with a verdict:
    - continue
    - narrow
    - pivot
    - kill
