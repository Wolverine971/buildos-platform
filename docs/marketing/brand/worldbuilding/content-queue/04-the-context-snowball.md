# The context snowball

**Lane:** Brand + Personal versions · **Format:** LinkedIn post + visual (this one WANTS an image/animation); IG-able · **Status:** queued

**Hook (brand):** "Your first brain dump is a snowball the size of your fist. Six months later it moves under its own weight."

**The idea:** THE canonical explainer for "context compounds" — word-hook + image-hook shipped together for the first time. Start small: one dump, a few notes. Every dump, task, and update packs on. Eventually the snowball is a force: agents brief you instead of interrogating you.
