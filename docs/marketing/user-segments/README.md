## Recommended Workflow

1. Start with [BuildOS Marketing Strategy 2026](../strategy/buildos-marketing-strategy-2026.md)
2. Pull tone and founder constraints from [DJ's Founder Context](../social-media/FOUNDER_CONTEXT.md)
3. Pull language patterns from customer-language docs
4. Write or revise segment messaging docs
5. Adapt for lifecycle or channel-specific docs like the [BuildOS Welcome Sequence](../strategy/buildos-welcome-sequence.md)

## Organizational Rules

- Put reusable segment messaging here.
- Put raw language research in `../customer-lingo-*.md`.
- Put platform execution in `../social-media/`.
- If a segment doc starts containing raw research dumps, move that research back into a customer-language doc.
