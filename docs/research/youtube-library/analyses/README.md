# Analyses

Canonical home for cleaned analysis notes derived from YouTube transcripts.

Use this naming pattern:

```txt
YYYY-MM-DD_<skill-or-topic-slug>_analysis.md
```

Analyses should focus on reusable operating lessons:

- what an agent should do differently
- what workflow or decision rule the source teaches
- what failure mode the lesson prevents
- which public skill guide or blog draft it may become
