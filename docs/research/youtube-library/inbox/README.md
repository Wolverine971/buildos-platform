# Inbox

Temporary staging area for new YouTube URLs, pasted transcripts, downloaded transcript files, analysis notes, or mixed source bundles that have not been normalized yet.

Use this folder only when a source is too messy to place directly in `../transcripts/` or `../analyses/`.

Before leaving work:

- identify the YouTube video ID when possible
- add or update the source row in `../INDEX.md`
- add standard frontmatter if the file is Markdown
- set `processing_status: needs_classification`, `needs_transcript`, `needs_analysis`, or `needs_synthesis`
- move clean transcripts to `../transcripts/` and clean analyses to `../analyses/` when they are ready

Do not create skill drafts from files in this folder until the source has been classified and indexed.
