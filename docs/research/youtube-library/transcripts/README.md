# Transcripts

Canonical home for new YouTube transcripts.

Use this naming pattern:

```txt
YYYY-MM-DD_<creator-or-channel>_<video-slug>.md
```

Downloaded transcripts should preserve:

- video title
- video URL
- creator or channel
- channel URL when available
- upload date when available
- transcript body

Canonical transcripts in this folder should follow `../FRONTMATTER_SCHEMA.md` so the library can track whether each video has been analyzed, converted into a draft skill, published, or intentionally excluded.

Existing scattered transcripts are indexed in `../INDEX.md` and should only be moved here during deliberate cleanup or when a skill draft is created from them.
