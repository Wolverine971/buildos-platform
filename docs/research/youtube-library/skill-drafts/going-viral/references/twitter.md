# Twitter / X — Going Viral Reference

Read this file when working on Twitter / X specifically. The cross-platform `SKILL.md` covers the universal layer (hook archetypes, killer-script skeleton, six-checkpoint gate). This file covers what's _different_ on X and the practitioners who understand its physics.

## Read me first

X is the only major platform where **text-only outperforms video by ~30%**. The currency is reply chains, not watch time. The first 90–120 minutes after posting determine reach. A reply is worth ~150x a like, a retweet ~20x. External links are punished at ~30–50% reach reduction. Premium accounts get ~10x more reach than free.

These three facts together change everything: hooks land in 6–10 words, threads hold attention by escalating curiosity rather than by retention math, and "build-in-public" works because revenue tweets ARE proof artifacts, not promotional copy.

## Algorithm reality (2025/2026)

Not 2019. Verify against current sources before publishing.

**Three-stage pipeline (open-sourced 2023, refined since):**

1. **Candidate sourcing** — algorithm pulls ~1,500 candidate posts from the 500M-post pool
2. **Neural network ranking** — scores each candidate using engagement weights
3. **Heuristics + filtering** — final ordering applied

**October 2025 shift:** Elon Musk announced **Grok replaces the legacy recommendation system** — Grok reads every post and watches every video to match users with content. This is X's version of the "interest graph" (matching readers to content regardless of follow relationship), and it's now the dominant ranking layer for the For You feed.

**Engagement weights:**

- Reply: ~150x a like
- Retweet: ~20x a like
- A reply that gets a reply from the original author: weighted +75 (vs +0.5 for a like)
- Like: baseline (~0.5 weight)

**What works:**

- **Text-only posts outperform video by 30%.** X is the only major platform where this is true.
- **First 2 hours are critical.** Speed of engagement determines reach more than total engagement.
- **1–2 niche-relevant hashtags** can lift engagement ~21%; 3+ hashtags get penalized ~40%.
- **Premium gets ~10x reach.** With only ~0.26% of users on Premium, the boost is enormous in proportional terms — non-Premium creators need to land 4–8x the organic engagement to match a Premium account's natural reach.

**What doesn't work:**

- **External links** — penalized 30–50%. Free accounts have seen zero median engagement on link posts since March 2025.
- **Late engagement** — a tweet that takes off after 4 hours is rare. The first 90–120 minutes define everything.
- **Long threads with weak T1** — if the first tweet doesn't land, the rest of the thread dies.

## The X content map

Five units of work, each with its own physics:

| Unit                       | When to use                                    | Hook architecture                                             | Length              |
| -------------------------- | ---------------------------------------------- | ------------------------------------------------------------- | ------------------- |
| **Single tweet**           | Aphorism, observation, hot take, status update | One sentence, surprise on the verb                            | ≤280 chars          |
| **Thread**                 | Teaching, narrative arc, framework, case study | T1 = 5-part intro compressed; T2-N = Value Loops with rehooks | 5–25 tweets         |
| **Atomic essay**           | Deep idea compressed                           | Image-formatted text post (screenshot of writing)             | 1 image + 1 caption |
| **Build-in-public update** | Revenue, milestone, scrappy demo, lesson       | Specific number / artifact in T1; transparency over polish    | 1–3 tweets          |
| **Quote-with-take**        | React to news, add POV                         | Acknowledgment + sharp angle                                  | 1 tweet             |

## Hook patterns that work on X

X hooks are sentences, not videos. The slot-fill problem still applies (Kallaway _100 Viral Hooks_) — pick the archetype, write the contrast — but compressed to one line.

**Six archetypes, X-flavored:**

- **Fortune Teller** — "In 18 months, the [X] industry won't exist." (Naval-style + future tense.)
- **Experimenter** — "Just shipped a [thing]. Here's what happened in the first 24 hours: [number]." (Pieter Levels build-in-public.)
- **Teacher** — "I spent 5 years failing at [thing]. Here's what finally worked." (Justin Welsh atomic intro.)
- **Magician** — Visual stun via image, screenshot, or specific number that breaks the scroll.
- **Investigator** — "Nobody is talking about [X], but it's already changed [Y]."
- **Contrarian** — "Everyone says [X]. They're wrong. Here's why."

**Specific patterns observed in 1M+ engagement posts:**

- **Specificity-as-hook.** Numbers, names, dates, dollar amounts. "$4.1M ARR" beats "huge growth."
- **Identity claim + counterintuitive predicate.** "I'm a [identity]. I [unexpected behavior]." (Naval, Pieter Levels.)
- **Single-line aphorism with surprise on the verb.** Naval's "How to Get Rich Without Getting Lucky" tweetstorm is the canonical example — every tweet stands alone, every verb is the punch.
- **Topic-as-headline.** "Twitter threads die when [specific failure mode]." First word names the topic; comma-claims do the contrast.

## Threads — Sahil Bloom's system

Sahil Bloom went from ~0 to 1M followers writing 225 threads over 26 months. Three structural patterns from his work (extract; verified via _How I Write_ with David Perell):

**1. Thread as a single argument, not a list.**

Most failed threads are "10 tips for X" with no connecting argument. Bloom's threads make one argument and use 8–15 tweets to build evidence for it. Each tweet either advances the argument or earns the next.

**2. T1 must work as a standalone tweet.**

If T1 is "🧵 a thread on X," nobody reads. Bloom's T1 reads as a full statement someone could screenshot — usually a counterintuitive claim + a promise.

**3. The "magnetic middle."**

Tweets 4–8 of a 12-tweet thread carry the weight. T1 hooks; T2-3 set up; T4-8 deliver the content; T9-10 escalate stakes; T11-12 close the loop and CTA.

Bloom's "secret weapon": volume of attempts, not formula. He published one long-form thread per week for 26 months. Some hit, most didn't. The data taught him what worked. **For a founder who can't publish weekly: skip threads. Use single tweets + atomic essays.**

## Aphorism architecture — Naval

Naval Ravikant's "How to Get Rich Without Getting Lucky" tweetstorm is the platonic ideal of an X-native idea-format: every tweet is a complete thought, the order matters but the parts don't depend on each other, the whole is a worldview.

Three properties:

- **Each tweet is sharable in isolation.** No "see thread for context."
- **Each tweet uses a surprising verb or noun pairing.** "Seek wealth, not money or status." The contrast is built into the sentence.
- **The set is a worldview, not a list.** Reading all 40 tweets installs a way of thinking, not a checklist.

When to use this format: you have a developed worldview on something, you're willing to commit to a single positioning for years, and you're optimizing for **evergreen reach** (tweets that get linked years later) over weekly engagement.

## Atomic posts — Justin Welsh

Justin Welsh has been writing daily on LinkedIn since 2018 and brought the same atomic-post discipline to X. Three rules from his work:

**1. One idea per post.** No multi-thesis posts. If you have two ideas, that's two posts on different days.

**2. The opening line is the entire game.** Welsh's posts open with one of: a contrarian claim, a specific number, a personal admission, or a question with a surprising frame.

**3. The post is an essay compressed.** Welsh's atomic essays follow a mini-killer-script structure: one-line hook → 2–3 lines of common belief / contrarian take → 4–6 lines of body (often a list of 3) → one-line close.

**Cross-platform note:** Welsh atomizes one idea across LI / X / email. Each platform gets a _native_ version, not the same post pasted three times. The LinkedIn version is longer and uses line breaks; the X version compresses to ~280 chars; the email version uses the post as a section inside a longer letter.

## Build-in-public — Pieter Levels

Pieter Levels' Twitter is the canonical build-in-public account. From the Lex Fridman #440 transcript:

**1. Revenue tweets are proof artifacts, not promotional copy.** Levels: "I tweet always like how much money these websites make." The number is the post. No commentary needed.

**2. Embrace imperfection publicly.** He acknowledges his AI image generator "is really bad half the time." Honesty paradoxically builds trust because it inverts the expected sales-mode polish.

**3. Do things that don't scale, publicly.** Early Photo AI: Stripe link + Typeform, no automation, manual processing of every order. The scrappy version is the brand asset. He documented every step.

**4. Problem-first positioning.** He doesn't market features; he markets problems his life surfaced (digital nomad → Nomad List, AI image friction → Photo AI). The post format: "I had this problem. Here's what I built. Here's what happened."

**5. Public accountability as distribution.** "12 startups in 12 months" was a public commitment. Levels: "I put it public, that I actually had to do it." Public commitment generates both accountability AND attention — the pre-announcement is itself viral.

**Founder voice translation:** for BuildOS, the equivalent moves are: tweet specific user counts / revenue numbers as they hit milestones; share scrappy pre-launch demos with their flaws visible; document problems-encountered-this-week posts; commit publicly to ship dates.

## Reply game

Replies are weighted ~150x likes. The reply game is half the X game.

**Pattern A — Steel-manning replies on bigger accounts.** Add genuine substance to a thread by a 100K+ account. If your reply gets a reply from the OP (+75 weight), your account gets dragged into the OP's reach.

**Pattern B — Sharp angle on news.** Quote-tweet news with a non-obvious angle. The quote-tweet keeps the news in the algorithmic stream while adding your POV.

**Pattern C — Controlled disagreement.** Disagree with the substance, never the person. The X discourse rewards "polite disagreement with receipts" because it generates reply chains without rage-baiting.

**Avoid:** Generic "great post!" replies (zero signal). Pile-on replies on controversial threads (signal poisoning). Copy-paste reply automation (banned + visible).

## Threading vs single tweet — decision rule

| Have                                                | Use                                                     |
| --------------------------------------------------- | ------------------------------------------------------- |
| One sharp aphorism with a surprising verb           | Single tweet                                            |
| One claim + 5–8 supporting points + an argument arc | Thread (8–12 tweets)                                    |
| One image of compressed prose                       | Atomic essay (image post)                               |
| One number / milestone                              | Build-in-public single tweet with the number front      |
| One news event + your angle                         | Quote-tweet with reply chain                            |
| Multi-paragraph thinking that's still loose         | Don't post yet — write it as a blog first, then atomize |

If in doubt: **single tweet > thread.** Threads are graveyard fodder when T1 doesn't land. Single tweets compound — Naval's evergreen tweets still drive traffic 5 years later.

## Anti-patterns specific to X

- **Posting external links without preamble.** ~30–50% reach loss; free accounts get ~zero median engagement on link tweets since March 2025.
- **3+ hashtags.** ~40% engagement penalty.
- **Long thread with weak T1.** Reach dies at T1; rest of the thread is invisible.
- **"🧵" emoji as the only T1 signal.** Without a standalone-readable claim in T1, the thread doesn't earn the swipe.
- **Quote-tweeting yourself to "boost" old posts.** Algorithm reads it as low-value. Better: rewrite the post fresh with new framing.
- **Late engagement chasing.** Posts that haven't moved in 90 minutes won't recover. Don't pin them, don't bump them — write the next one.
- **Generic "agree?" closer.** Treats engagement as transactional. Ask substantive questions or close with a stance.
- **Copying Naval / Welsh / Bloom's exact phrasing.** Pattern-matching is fine; sentence-matching is not. The algorithm penalizes near-duplicate phrasing.
- **Daily threads on the same topic with no variance.** Audience-match becomes too narrow; subsequent posts get penalized as the avatar gets oversaturated.

## When this is the wrong platform

- **Long-form ideas that need >1500 words to develop.** Use blog + tweet a quote-card.
- **Visual-heavy work.** Instagram or TikTok, not X.
- **B2B enterprise sales.** LinkedIn has 10x the conversion physics for that audience.
- **Local-business / service intent.** Wrong platform — service hub pages and local SEO win.
