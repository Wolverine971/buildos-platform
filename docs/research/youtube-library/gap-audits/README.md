# Skill Gap Audits

This folder stores gap audits for standalone skills, public agent-skill blog drafts, runtime skills that should not be edited directly, and other source files that do not have a natural sibling audit location.

Use `/skill-gap-audit` to create or update these audits.

For category-level skill combos, prefer sibling audit files in `docs/research/youtube-library/skill-combo-indexes/`, such as:

```txt
docs/research/youtube-library/skill-combo-indexes/FOUNDER_OPS_AND_CAREER_GAP_AUDIT.md
```

Gap audits are research-planning artifacts. They should identify blind spots, missing sources, expert search targets, and draft readiness before a skill is written or revised.
