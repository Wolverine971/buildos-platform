## Which resume to send

- **`David_Wayne_Resume_2page.md`** is the version to actually send to recruiters. It's the polished, AI-hallmark-scrubbed (no em-dashes, no AI-flavored phrasing), 2-page-target build. Paste into Google Docs and export to PDF.
- **`David_Wayne_Resume.md`** is the longer master with extra detail. Keep as the source of truth for content. When tailoring for a new role, copy this and trim.

## When applying for a new role

1. Copy `David_Wayne_Resume.md` to a new working file.
2. Tailor the Summary and reorder Experience bullets to match the JD.
3. Run a light AI-hallmark trim pass (em-dashes, "leverage", "robust", "comprehensive", etc.).
4. Save the JD next to your tailored resume under `job-applications/<company>-<YYYY-MM>/`.
5. Keep the 2-page target unless the role explicitly invites longer.
